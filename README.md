# PoojaWebSite

Static website repository for Pooja. This repo contains HTML/CSS/JS assets for a simple site.

## Structure
- `index.html` — main page (add your content)
- `assets/` — images, styles, scripts

## Development
Open `index.html` in a browser or serve locally:

```sh
python3 -m http.server 8080
```

Then visit http://localhost:8080.
