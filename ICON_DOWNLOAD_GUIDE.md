# Reliable Free Contact Icon Download Commands
# From: Simple Icons (MIT License) - Official brand icons

## Download WhatsApp Icon (PNG format - 200x200px)
curl -o whatsapp-icon.png "https://raw.githubusercontent.com/simple-icons/simple-icons/develop/icons/whatsapp.svg" && \
convert -background none whatsapp-icon.svg -resize 200x200 whatsapp-icon.png

## Download Telegram Icon (PNG format - 200x200px)  
curl -o telegram-icon.svg "https://raw.githubusercontent.com/simple-icons/simple-icons/develop/icons/telegram.svg" && \
convert -background none telegram-icon.svg -resize 200x200 telegram-icon.png

## Download Gmail/Email Icon (PNG format - 200x200px)
curl -o email-icon.svg "https://raw.githubusercontent.com/simple-icons/simple-icons/develop/icons/gmail.svg" && \
convert -background none email-icon.svg -resize 200x200 email-icon.png

# ALTERNATIVE: Font Awesome Free Icons (CC BY 4.0 License)

## Download WhatsApp Icon
curl -o whatsapp-icon.svg "https://raw.githubusercontent.com/FortAwesome/Font-Awesome/6.x/svgs/brands/whatsapp.svg" && \
convert -background white whatsapp-icon.svg -resize 200x200 whatsapp-icon.png

## Download Telegram Icon
curl -o telegram-icon.svg "https://raw.githubusercontent.com/FortAwesome/Font-Awesome/6.x/svgs/brands/telegram.svg" && \
convert -background white telegram-icon.svg -resize 200x200 telegram-icon.png

## Download Envelope/Email Icon
curl -o email-icon.svg "https://raw.githubusercontent.com/FortAwesome/Font-Awesome/6.x/svgs/solid/envelope.svg" && \
convert -background white email-icon.svg -resize 200x200 email-icon.png
